
**PROJECT INSTRUCTIONS FOR RUNNING**

1. Compile the source file with gcc

* `gcc -o project3 project3.c`


2. Run the object file with any of the following formats:

Usage: 
* `./project3 create <new_idx_filename>`
* `./project3 insert <existing_idx_file> <new_integer_key> <integer_value>`
* `./project3 search <existing_idx_file> <integer_key>`
* `./project3 load <existing_idx_file> <existing_csv_file>`
* `./project3 print <existing_idx_file>`
* `./project3 extract <existing_idx_file> <new_csv_filename>`


3. Done!
